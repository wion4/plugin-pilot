# Third-Party Plugin Disclaimer — plugin-pilot

**Last updated:** April 4, 2026

## Important Notice

Plugin Pilot can search for and suggest installation of third-party plugins and skills from sources outside the official Anthropic marketplace, including but not limited to:

- Public GitHub repositories
- Unofficial/custom plugin marketplaces configured by the user

## No Warranty or Guarantee

Third-party plugins are **not reviewed, verified, endorsed, or supported** by the plugin-pilot authors or Anthropic. By installing a third-party plugin, you acknowledge and agree that:

1. **No security guarantee** — Third-party plugins may contain malicious code, vulnerabilities, or backdoors. Plugin Pilot performs only basic structural verification (presence of manifest, license, README) and cannot detect malicious behavior.

2. **No functionality guarantee** — Third-party plugins may not work as described, may cause errors, data loss, or system instability.

3. **No compatibility guarantee** — Third-party plugins may conflict with other installed plugins, Claude Code updates, or your system configuration.

4. **No privacy guarantee** — Third-party plugins may collect, transmit, or store your data in ways not disclosed or contrary to their stated privacy policies.

5. **No liability** — The plugin-pilot authors accept **no responsibility or liability** for any damage, data loss, security breach, or other harm resulting from the installation or use of any third-party plugin.

## User Responsibility

By using Plugin Pilot's community search and third-party plugin installation features, you agree to:

- Review the source code and documentation of any third-party plugin before installation
- Accept full responsibility for any consequences of installing third-party plugins
- Report any malicious or harmful plugins to the respective platform (GitHub, etc.)

## Trust Score

Plugin Pilot provides a "trust score" for community plugins based on structural completeness (presence of manifest, license, README, proper directories). This score is **not a security assessment** and should not be interpreted as an endorsement or guarantee of safety. A high trust score only indicates proper plugin structure, not the absence of malicious code.

## Official Marketplace

Plugins from the official Anthropic marketplace (`claude-plugins-official`) have undergone Anthropic's review process. Plugin Pilot always prioritizes official marketplace plugins over community alternatives.
